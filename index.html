Devlog for a custom-engine city builder made in Rust + WGPU

Devlog #1 - Six Months In

22 April 2026

After six months of development, Rusty Skylines is starting to feel like a real game.

Main progress so far:

  • Custom UI system made from scratch
  • Blur behind UI panels
  • Zoning and lot placement systems
  • Chunk-based world architecture
  • Traffic simulation groundwork
  • Many engine rewrites and improvements

The hardest part has been building core systems properly instead of rushing features.

Next goals:

  • Citizens
  • Road improvements
  • Economy systems
  • More polish

Thanks for reading.
